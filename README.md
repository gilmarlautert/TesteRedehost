# TesteRedehost

 - Instale o .NET CORE em: https://www.microsoft.com/net/core

 - Instale o Node.js + npm: https://docs.npmjs.com/getting-started/installing-node

 - Restore do projeto

  ```
    dotnet restore
  ```

 - Adição dos pacotes NPM

  ```
    npm install
  ```  

 - Execução do projeto  

  ```
    dotnet run
  ```

 - Execução dos testes  

  ```
    dotnet test
  ```
